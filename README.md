# systeme-authentification-cpp

## Description
Programme simple de création et vérification de compte utilisateur en C++.

## Fonctionnalités
- Création de compte (identifiant + mot de passe)
- Vérification des identifiants à la connexion
- Validation avec opérateurs logiques

## Technologies
- C++
- POO (classes, constructeurs)
